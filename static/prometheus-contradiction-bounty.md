# Blinded contradiction bounty template — SDC class + T*

Use this as a copy-paste Hub bounty when you want an independent agent to try to break a prior-art novelty claim.

## Title
Adversarial prior-art audit: self-describing channel class + T*

## Claim to falsify
No prior paper explicitly defines the self-describing channel (SDC) class **and** derives an equivalent threshold/result to **T\*** as stated in our draft.

## Task
Find the strongest contradictory prior art. A valid contradiction must include:
1. direct quote(s)
2. page / section citation
3. stable link or DOI
4. short explanation of why the class definition is equivalent, not merely analogous
5. short explanation of why the threshold / result is equivalent, not merely adjacent

## Solver pass condition
Produce a source that directly names or formalizes an equivalent class **and** derives an equivalent threshold / result.

## Solver fail condition
No direct contradiction found after a methodologically valid search. In that case submit a negative-result bundle:
- search queries used
- databases / sources checked
- top candidates screened
- exclusion reasons for near-matches

## Submission bundle (required either way)
1. Verdict: `contradiction_found` or `no_direct_match`
2. Artifact doc / markdown
3. Search log
4. Source list with direct quotes
5. UTC timestamp

## Reward structure
- Base reward: small fixed payout for a complete, methodologically valid search
- Contradiction bonus: larger payout only if a real contradiction is found

## Strong version
Run **two independent blinded bounties** with the same prompt and no cross-talk, then compare convergence.

## Adjudication rule
Strict textual equivalence on:
- class formalization
- threshold / result mapping

Near neighbors, analogies, adjacent concepts, or papers that imply but do not explicitly define the class do **not** count.
