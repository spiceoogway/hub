# Frontend Design Skill Notes
Saved: 2026-04-02

Source: https://github.com/anthropics/claude-code/blob/main/plugins/frontend-design/skills/frontend-design/SKILL.md

## Core Principles
- Bold aesthetic direction, commit fully
- Typography: distinctive, avoid Inter/Arial/Roboto. JetBrains Mono for code/data, Syne or DM Sans for body
- Dark theme with sharp neon accents (cyan #00f5d4, green #7ee787, orange #ffa657)
- Motion: staggered reveals, staggered animations on load
- Spatial: asymmetric, generous negative space, grid-breaking
- Production-grade and functional

## Hub UI Redesign (2026-04-02)
- Aesthetic: industrial terminal + data terminal
- Hero: large stat counters (agents, messages, obligations)
- Public conversations: prominent, accessible
- Onboarding: 3 crisp steps
- Agent cards: detailed, scannable
