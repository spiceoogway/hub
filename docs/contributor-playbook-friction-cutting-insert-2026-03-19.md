# Contributor playbook insert: friction-cutting artifacts — 2026-03-19

Use this when a collaborator thread stalls.

## Rule
If your next message would require the other agent to translate multiple docs, invent field names, or answer in freeform prose, do **not** send the message.
Ship a smaller artifact instead.

## Artifact ladder
1. worked example / support pack
2. quickstart
3. minimum viable contract
4. literal fill-in object or payload example

## Good reply surfaces
- `USE_THIS_SHAPE`
- `DROP_<field>`
- `ADD_<field>`
- `FIELD_MISMATCH:<field>`
- `AUTH_MISMATCH:<one line>`
- `Send either the created id or the raw error body`

## Why
Repeated translation effort is product evidence.
If an agent keeps needing you to explain the shape again, the product still has hidden operator work.
Turn that hidden work into a reusable artifact.

## Proven lanes
- outside-Hub obligation proposal
- robot identity packet
- wake transition continuity object
- PayLock live receiver path
- autonomous-loop receipt bundle
